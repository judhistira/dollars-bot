# Dollars Indonesia Community Bot - Dokumentasi Proyek

## Deskripsi
Bot Discord yang menyemangati warga komunitas "Dollars Indonesia" dengan pesan-pesan puitis, dingin, dan introspektif ala karakter Rangga di film "Ada Apa Dengan Cinta".

## Struktur Proyek
Proyek ini memiliki struktur modular untuk kemudahan pengelolaan:
- **`index.js`**: Entry point aplikasi. Menjalankan server Express dan menangani webhook.
- **`bot.js`**: Logika inti bot, memproses dan mengirim pesan.
- **`gemini.js`**: Mengelola interaksi dengan Google Gemini API.
- **`utils.js`**: Kumpulan fungsi utilitas.
- **`test-console.js`**: Skrip untuk tes output di konsol.
- **`test-reminder.js`**: Skrip untuk tes pengiriman pesan ke Discord.
- **`.env.example`**: Contoh file environment.
- **`package.json`**: Dependensi dan skrip proyek.
- **`README.md`**: Dokumentasi utama.

## Konfigurasi Environment Variables
- `DISCORD_TOKEN`: Token bot Discord.
- `GEMINI_API_KEY`: API key Google Gemini.
- `NEWS_API_KEY`: API key NewsAPI.org (opsional).
- `CHANNEL_IDS`: ID channel Discord tujuan (pisahkan dengan koma jika lebih dari satu).
- `GEMINI_MODEL`: Model Gemini yang digunakan (e.g., "gemini-1.5-flash").
- `TIMEZONE`: Zona waktu untuk bot (e.g., "Asia/Jakarta").

## Cara Kerja
1.  Layanan cron job eksternal memanggil endpoint webhook (`/dollars-reminder`) di `index.js`.
2.  `index.js` menerima panggilan dan memicu `sendEncouragementMessage` dari `bot.js`.
3.  `bot.js` menjalankan logika utama:
    a.  Menggunakan `getTimeInfo` dari `utils.js` untuk menentukan waktu.
    b.  Memanggil `getGreetingMessage` untuk membuat sapaan.
    c.  Memanggil `generateFinalMessage` dari `gemini.js` untuk membuat konten pesan utama dengan AI.
    d.  Menggunakan `splitMessageIntoChunks` dari `utils.js` jika pesan terlalu panjang.
    e.  Mengirim pesan ke channel Discord.
4.  `gemini.js` berisi semua detail prompt yang dikirim ke Google Gemini API.

## Testing
- **Tes Konsol**: `npm run test:console`
- **Tes Pengingat**: `npm run test:reminder`
